# UNO 在线联机 - 页面结构与组件

## 页面路由


| 路由               | 页面  | 说明                        |
| ---------------- | --- | ------------------------- |
| `/`              | 首页  | 昵称设置 + 创建房间/加入房间          |
| `/room/[roomId]` | 房间页 | 玩家列表 + 等待 + 房主操作          |
| `/game/[roomId]` | 游戏页 | 牌局进行 + 结算（根据 status 切换视图） |


## 页面流转

```
首页 → 创建房间 → 房间页
首页 → 加入房间 → 房间页
房间页 → 开始游戏 → 游戏页
游戏页（结算）→ 开始下一局 → 游戏页
游戏页（结算）→ 结束游戏 → 首页
```

### 组件


| 组件             | 说明              |
| -------------- | --------------- |
| NicknameInput  | 昵称输入框 + 随机生成按钮  |
| CreateRoomForm | 庄家选择方式 + 创建房间按钮 |
| JoinRoomForm   | 房间号输入 + 加入按钮    |

### 组件


| 组件              | 说明                        |
| --------------- | ------------------------- |
| RoomHeader      | 房间号 + 复制链接按钮 + 庄家方式       |
| PlayerList      | 玩家列表（房主标记、AI 标记、移除按钮）     |
| HostActions     | 添加 AI 按钮（仅房主可见）           |
| StartGameButton | 开始游戏按钮（仅房主可见，人数 >= 2 时可点） |

## 游戏页 `/game/[roomId]`

根据 status 切换不同视图。

### 选庄视图（status = choosing_dealer）

```
┌─────────────────────────────────┐
│  选庄 - 摸牌比大小               │
├─────────────────────────────────┤
│  小明:  [7]                     │
│  小红:  等待摸牌...              │
│  机器人A: [3]                   │
│                                 │
│         [ 摸牌 ]                │
└─────────────────────────────────┘
```

```
┌─────────────────────────────────┐
```

### 组件


| 组件              | 说明                               |
| --------------- | -------------------------------- |
| GameHeader      | 局数、摸牌堆数量、出牌方向、当前玩家               |
| OpponentArea    | 其他玩家信息（昵称、手牌数量、UNO 标记）           |
| DiscardPile     | 弃牌堆顶牌 + 当前指定颜色                   |
| MyHand          | 自己的手牌展示（可点选）                     |
| ActionButtons   | 操作按钮组（出牌/摸牌/跳过/接受/质疑，根据状态显示不同按钮） |
| ColorPicker     | 颜色选择弹窗（万能牌时显示）                   |
| DealerSelection | 选庄摸牌界面                           |
| DealButton      | 发牌按钮（dealing 状态，仅庄家可见）           |
| RoundResult     | 结算界面（得分表 + 下一局/结束按钮）             |
| PauseOverlay    | 断线暂停遮罩（倒计时显示）                    |
| UnoAlert        | UNO 提示效果                         |
